== Description ==

Its Awesome. Only 1 Plugin and have 2 widgets. you do not need any shortcode and customization.
Just Install and Enjoy 2 Dashboard Widgets.

A few notes about the sections above:

* __User Status__ - Show your all user like as admin, administator, author, subscriber. and edit them.
* __Todo List__    - Add, edit, delete your list

**Features**
WP Dashboard Eathim has many feature.

* Includes 2 awesome Dashboard widgets

**Privacy**
It does not set any coockies. and does not links any third party location. this plugin does not collect any data of user.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wp-dashboard-eathim` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Dashboard->Plugin Name screen to configure the plugin
